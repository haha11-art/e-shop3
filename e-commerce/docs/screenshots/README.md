# 前端异步请求功能 - 测试截图说明

## 📸 截图获取指南

本文档说明如何获取任务要求的接口测试截图。

### 前置准备

1. **启动前端服务**
   ```bash
   cd D:\e-commerce\frontend
   npm run dev
   ```
   前端将在 `http://localhost:5175` 运行

2. **打开浏览器开发者工具**
   - 访问 `http://localhost:5175/products`
   - 按 `F12` 或右键 → "检查"
   - 切换到 **Console** 标签（用于查看请求日志）
   - 切换到 **Network** 标签（用于查看HTTP请求，API模式时需要）

##  截图技巧

1. **使用浏览器截图工具**
   - Chrome/Edge: `Ctrl + Shift + I` 打开开发者工具 → 右上角三个点 → "Capture screenshot"
   - Firefox: `Ctrl + Shift + S` 直接截图

2. **确保截图清晰**
   - 缩放比例设置为 100%
   - 截取完整的 Console 日志区域
   - 必要时放大字体（Console 设置 → Font size）

3. **标注关键信息**
   - 用红框标出控制台的关键日志
   - 用箭头指示操作流程
   - 添加文字说明验证点

4. **命名规范**
   - 文件名与文档中一致（如 `mock-page1.png`）
   - 保存到 `D:\e-commerce\docs\screenshots\` 目录
